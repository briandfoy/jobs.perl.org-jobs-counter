Role Description:
Perl / Web UI Developer for a leading global investment bank. This is a new
team formed of experienced engineers from a number of application hosting
platforms. While successful, each hosting platform had varying levels of
maturity in automation, self service and application management concepts.
The goal is to standardize application hosting infrastructures through the
implementation of a modular platform which provides a rich interface to
service teams. In other words, putting in place a framework for software
experts to provide their software as a service.

The first release of the platform has been designed around one of the most
complex services, and already has a high degree of separation between
management of core service hosting processes and the software specific
processes. They are looking to mature this and extend the core feature set
to bring on board more services.

The platform will be providing the framework as a set standard interfaces
to service specialist teams which in turn host and manage a large number of
applications, many of which will be business critical.

Portal Framework Engineers are responsible for designing and implementing
