Description:
The Research Computing Center (RCC) at the University of Chicago seeks a
Web Developer to build and manage the unit website, develop tools that
enable remote account management, and tie together the RCC high-performance
computing resources to the user facing site. The Web Developer will also
consult with researchers to give them advice in the development of web
interface features for their research performed on RCC resources. 

Position Responsibilities 
- Manage the development of the RCC website 
- Interpret user-interface and functional design elements requests from a
team of technical staff members 
- Build tools to enable integration of the RCC website with existing and
future resources used for high-performance computing 
- Develop tools and interfaces to create and manage accounts and permit
users to manage and display their account information 
- Advise RCC on technologies that intersect high-performance computing and
web technologies such as connecting to cloud computing interfaces with REST
APIs 
- Advise and/or help researchers create a web presence for research
performed using RCC resources

Required skills:
A minimum of three years experience developing websites in a professional
environment required. 
Experience in web site and web-based application development and support
required. 
Experience with HTML/XHTML and CSS required. 
Experience with user interface design required. 
Web programming experience with frameworks written in languages such as
PHP, ASP, JSP, Python, Perl, and/or Ruby required. 
Experience developing and designing websites and web management tools in a
Unix based operating environment required. 
Experience with relational databases such as MySQL or PostgreSQL required. 

Experience with directory services such as LDAP or Microsoft Active
Directory required. 
Experience with revision control software required. 

Experience with web service consumption and RESTful API preferred.
