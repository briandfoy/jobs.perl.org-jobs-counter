OPPORTUNITY:
Lions Bay Media is looking for a full time DevOps Programmer to join our
team. We offer a casual, professional work environment and corporate
culture. We strive to take advantage of the latest state-of-the art tools
and techniques to deliver superior technology. Our systems are powered by
advanced open source components.

RESPONSIBILITIES:
This is a unique position which comprises 2 distinct roles. We're looking
for an individual with a broad skills base who can tackle systems
administration challenges as well as build great software.

1. System and Network administration/automation
-Build, configure and deploy production servers (including virtualization)
-Develop automation systems
-Maintain network infrastructure
-Maintain PCs and other devices
-Troubleshoot hardware and software
-Ongoing performance and reliability monitoring

2. Software Development
-Design, develop, and deploy software in a production environment
-Software maintenance, enhancement, diagnostics and bug fixing
-Write technical documentation
-Integrate systems with third-party APIs
-Write unit tests, perform manual testing
-Work closely with QA
-Front end development including HTML/CSS and user interface design

DESIRED QUALIFICATIONS:
-Degree or diploma in Computer Science (or a related field) or equivalent
education/experience
-Previous professional experience in systems administration/automation, and
ideally software development as well
-Comfortable with virtualization technologies such as KVM, Xen, etc.
-Strong Linux/Unix skills
-Competent with system automation tools such as Vagrant, Chef, Puppet, etc.

-Strong knowledge of core web technologies and server software such as
SMTP, DNS, SSH, SSL/TLS, etc.
-Familiar with database servers and key-values stores including Postgres,
MySQL and Redis
-Knowledge of cloud services such as Amazon web services
-Troubleshooting and repair of server/PC hardware
-Proficient with networking hardware and topology
-Knowledge of PCI/DSS would be a big plus
-Knowledge of several programming languages such as Perl, JavaScript, Ruby,
PHP, etc. (Although we're not a PHP shop, we do have several legacy PHP
applications) 
-Comfortable with SQL and relational database design
-Version control systems (especially git)
-Comfortable with web application frameworks (such as Mojolicious) and
related technologies (RPC methods such as JSON-RPC and SOAP would be
useful)
