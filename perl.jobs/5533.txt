Compuware Corporation, a Software Products and IT Professional Services
company, has the following position available in sunny southern
California:
=20
Web Administrator

The primary responsibility of this position is to perform application
administration on web and application servers. Includes managing access
control on Extranet and Intranet web servers, interfacing with Unix and
Network Administration teams to negotiate backups and software upgrades.


=20

Perform routine analysis of web server and application logs to
proactively detect problems on website. Assist in troubleshooting web
applications, and maintaining quality of service/performance. Keep
server software current -- apply or recommend upgrades to servers.
Install Apache/iPlanet/WebLogic servers as needed, write application
monitoring programs, application stability scripts, ksh shell scripts,
wlst shell scripts for Weblogic monitoring, manage internal Digital
Certificate Authority, support BEA WebLogic developers on J2EE projects,
manage log files for public and extranet Web sites, provide log data
feeds to NetIQ/WebTrends, and supply technical advice.

=20

Support application developers during application deployment and
testing. Deploy and configure custom applications in Production. Perform
capacity testing on new applications. Generate website usage reports.
Typically background includes a Bachelors degree with computer related
focus or the equivalent and 5+ years related experience.=20

=20

Solid experience of Unix/Linux operating systems. General understanding
of internetworking principles, devices and troubleshooting tools. Sound
understanding of hardware components. Demonstrated experience
administering high availability commercial web servers. Clear
understanding of general security concepts and n-tier architecture.
Server side support. Will be part of a dev team.

=20

Technical environment: Web Logic and Sun moving to Linux
