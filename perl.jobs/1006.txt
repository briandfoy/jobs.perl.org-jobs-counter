Description:
Our team designs, builds, and hosts sophisticated high-end eCommerce
solutions for clients on an outsource basis.  We are in need of a Senior
Web Development Engineer and a Senior QA Engineer to join our talented
team.  These positions are available immediately, and will be
contract-to-perm positions.  

Required skills:
Domain Knowledge:

Our Web Development and QA Engineers are key to delivering a quality
product that meets the client’s needs.	 The ideal candidate will have real
world experience with 24x7, industrial strength eCommerce websites. 
Familiarity with shopping carts, cookies, credit card processing, digital
rights management, electronic software downloads (ESD), and electronic
licensing is a plus.  The QA position requires extensive experience as a
“White box” tester, being able to probe and evaluate the internals of a
software product.

Communication skills:

*Effective verbal and written communication skills are a must. The ability
to articulate design concepts to a variety of audiences including software
engineers, management, sales people, and of course clients.  Documentation
is also an integral part of our development methodology.   Web Development
Engineers must be able to write unit test plans based on technical
specifications.  QA Engineers must be able to write test plans based on
technical specifications, write effective bug reports, and participate in
code walk-throughs.

Technical skills:
*  Past experiences with systems that create dynamic web pages using an
environment like Perl/mod_perl, JSP, ASP, and ColdFusion or similar,
accessing a SQL database on the back end is vital.    
*  Specific Perl experience should include: OOA/OOD leading to mod_perl
implementations, focus on performance techniques, and use of common CPAN
modules.
*  Strong SQL skills should include:  optimization techniques, use of SQL
Tools such as SQL+.
*  We have extended the HTML tag set to support rapid deployment of
customized web interfaces.  Knowledge of HTML required, XHTML, CSS, and
JavaScript would be helpful.  
*  Much of our file transfer technology is based on XML/SOAP.  Working
knowledge of XML constructs, including DTD definition and XSLT validation
would be very valuable.  
*  Working knowledge of networking technologies, including security and
encryption on the Internet, and basic networking concepts. 
*  Hands on experience with Apache web servers is also a plus.	
*  Knowledge of and productivity with Unix is desirable.  
*  Experience with Internationalization and Localization efforts,
especially with multi-byte languages.
*  A B.S./M.S. in Computers Science or related major, with 3+ years
experience in Software Development.

Additional skills for QA Engineers:
*  Verify internal database values by using SQL based tools to query
various back end database components.  
*  File maintenance system tools such as CVS.
*  Bug tracking system tools such as Bugzilla.
*  Configuring and running automated test scripts.
