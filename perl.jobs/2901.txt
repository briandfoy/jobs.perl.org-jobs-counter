As a result of increased business, a leading Global Company is looking for
yet another Sr Perl Backend Systems/Database Developer on/into Oracle to
join a growing Data Operations team of Oracle Database Developers.  

The Sr Perl Backend Systems/Database Developer on Oracle will be
responsible for developing, enhancing, troubleshooting Company’s backend
Perl/Oracle database applications, in a UNIX (Solaris/Linux) environment,
using Perl, SQL, PL/SQL, SQL Loader and Unix shell scripts.  Company is
running several instances of Oracle 8i databases on Solaris/Linux, loading
100,000’s transactions/day of financial data into a 30+ terabyte Data
Warehouse. Transaction processing and reporting systems are developed in
Perl, Unix Shell Scripts, PL/SQL, SQL Loader and SQL. Candidate will be
responsible for developing and optimizing the backend systems for
stability, scalability and efficiency.	Responsible for designing and
optimizing database schema, queries, and complex stored procedures under
Oracle 8. Locate and fix bugs in the existing Perl, SQL and PL/SQL code
base in a timely manner.  Create and modify Unix shell scripts. Candidate
will also maintain the integrity of the databases in order to provide
accurate information.  Candidate will provide technical assistance by
responding to inquires from users regarding errors, problems or questions
about programs. Ensure availability of data in a timely manner. Consult
with users to design and code reports/programs to meet their needs. 
Candidate will be required to be on-call 1 week every 2 months.

The company offers full benefits (PPO & HMO) including dental and vision,
401K, Stock Options and Purchase Plans, 2 weeks vacation, 8 paid sick days,
Short and Long Term Disability, Life Insurance, Employee Assistance
Program, casual dress and flexible work hours that all start upon
employment.

REQUIREMENTS
MUST HAVE 3-4+ years developing large high-volume (50-100K
transactions/day) backend Systems in Perl utilizing SQL, SQL Loader, PL/SQL
and Unix shell scripts on Oracle
MUST HAVE Perl and Unix Shell Scripting experience, preferably on Solaris
and Linux
Ability to understand and create very complex SQL statements
Must be able to apply optimization methods, including Oracle Hints, to SQL
statements
Must have experience creating stored procedures and triggers with PL/SQL
Should have experience with multiple storage techniques (flat files, DB

Ability to extract unique records from Oracle using SQL and PL/SQL
Must be able to be on call 1 week every 2 months
PLUS:
Unix System Administration or DBA experience is a plus, but not necessary.

backend systems/database development at each employer will be considered.
