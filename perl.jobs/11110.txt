change management to the enterprise.  The user experience is a highly

Flash, Flex and HTML. The server side is Linux / MySQL with a Ruby on Rails
framework served up by Apache / Passenger. The backend consists of numerous
Perl-based engines that collect, model and analyze network data and
automate network operations. The development methodology centers around
Agile Scrum, using lightweight processes focused on delivering tangible
results quickly and often.


The Challenge:

The Server Engineer is responsible for the development, design, coding and
testing of network analysis and management features in the “back-end” of
the company’s “NetMRI” Linux-based appliance.
