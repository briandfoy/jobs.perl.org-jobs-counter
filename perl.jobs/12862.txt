Description:
As CLIQ Digital continues to grow we look for an experienced Perl Developer
as a part of our Technology Team.

The CLIQ multi-media services platform defines a vital network component
for the efficient distribution of mobile service content, forming an
inter-connection hub to all major gateways worldwide. The Perl backend
component of CLIQ’s architecture connects to more than 30 countries, and
this number is growing. The platform is built on top of a solid framework
of interconnected Perl modules and scripts within a distributed network of
powerful Linux servers. High-performance, robustness, redundancy and
scalability are important. The development team is continuously improving
this backend system by using state of the art open-source technology like
Perl, mod_perl, Apache2, XML, Json, MySQL, CPAN-style packaging,
object-oriented programming, continuous integration, module testing and
version control guide development. 

Job Description
Maintain, improve and enhance modules of a platform used to manage
subscriptions, bill and deliver content to mobile consumers worldwide. The
applicant must be able to work mostly independently on small projects
involving connections to SMS aggregators and implementation of new
features. The applicant will gather the relevant requirements, implement
them and cover the implementation with tests and documentation. The lead
developer will guide the applicant when dealing with larger refactorings
and/or the design of larger features that affect multiple modules.

To get a feeling for the platform the applicant will first learn to set up
services on the platform. Operational tasks will remain a small part of the
job, both as backup for the main application engineer and to promote an
awareness of the consequences of certain design decisions from an
operational point of view.

Core responsibilities
•	Problem analysis
•	Functional and technical design
•	Software development – object oriented
•	Module & system testing
•	Documentation, packaging and deployment

Required skills:

•	Analytical skills

•	Experience with software version control like	  CVS and/or SVN
