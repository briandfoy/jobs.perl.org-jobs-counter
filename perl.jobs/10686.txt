The DOE Joint Genome Institute in Walnut Creek, CA has an exciting

developers and analysts to maintain existing data management and QC systems

requirements, and implementation of systems to meet goals. Position reports

- Gather feedback and requirements to develop software to support the
existing production data management and QC systems, including the
resolution of custom user and data retrieval requests.

management and QC systems, as well as new versions of systems when
necessary.
- Create, manage, and query MySQL databases.
- Develop and utilize Perl classes, modules, and scripts for managing data
and performing analyses; and Perl/CGI web pages for providing user access.
- Provide responsive customer support, assisting users with the prompt
resolution of data entry errors, software malfunctions and other system
needs.

- B.S. in Computer Science or a related scientific or technical field (or

- Demonstrated experience utilizing Perl.
- Demonstrated experience with database programming and interfaces,

- Demonstrated success designing and implementing systems and software in a
