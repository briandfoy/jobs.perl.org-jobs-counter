The DOE Joint Genome Institute in Walnut Creek, CA has an exciting
Bioinformatics Software Developer opening available to support the
Sequencing Quality Assurance / Quality Control Group. Will develop software

software system using LAMP for analysis of short read sequence technology.

investigate improved methods for performing sequencing data analysis. Will
work independently and contributing as a member of a dynamic,
multidisciplinary team environment. Position reports to the QA/QC Group
Lead.

- Identify client needs and requirements to develop, implement, test,
integrate and deploy new and/or existing software applications and work
with other developers to coordinate work.
- Develop a software system for the analysis of short read sequence
technology.
- Create, manage, and query MySQL databases.
- Develop and utilize Perl classes, modules, and scripts for managing data
and performing analyses; and Perl/CGI web pages for providing user access.
- Interface with end-users to gather feedback and requirements.
- Provide responsive customer support; provide users with the prompt
resolution of data entry errors, software malfunctions, and other system
needs.
