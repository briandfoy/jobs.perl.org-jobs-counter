Software Tools Developer

This position in LogLogic’s LogLabs group is responsible for creating an
efficient tool-assisted workflow for our team of analysts to deliver
log-parsers to our LogAppliance.  You will rapid-prototype a
text-retrieval, database-backed system to support an existing process, as
well as design and implement validation and code-generation tools. You will
be working with analysts, architects, developers, quality engineers and
product managers to create the next generation of LogLogic products, and
learn from the experts in log and systems management.  

Responsibilities:
• Rapid prototype Twiki-based Workflow tools for our LogLabs procedures.
• Design and implement text analysis, source-code validation scripts and
code-generation tools.
• Implement text-retrieval, clustering and machine-learning algorithms on
data and documents.
• Support test automation using VMware with the SQA group.
• Build a robust, database-backed workflow support system.

Requirements: 
• B.S. Computer Science (or related field), M.S. preferred.
• 5+ years of software development in Unix/Linux environment.
• Ability to implement web applications using Perl, Python, Ruby or Java
frameworks, databases such as MySQL, and deployed on Linux.
• Collaborated with architects and designers to deliver prototypes quickly.
