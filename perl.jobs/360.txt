Description:
MyCity Networks is an Internet Infrastructure technology provider located
in downtown Miami. We have a unique work environment with health benefits,
stock options and paid parking.

We are looking for a System Administrator who will be able to assist in the
day to day management and monitoring of Linux, servers and Networking
equipment (routers, switches, load balancers, firewalls) at collocated
server farm. This includes monitoring of CPU and memory utilization, disk
capacity, network bandwidth and web site uptime. Installation of security
and software updates to operating system, network bios, etc. as needed.


Required skills:
* Serve as the main point of contact for the 24x7x365 monitoring of all
production web, app and database servers and storage systems. Includes the
administration of tools and software for monitoring web server uptime and
sending alert emails to text-based wireless communication devices

* Perform routine web, app and database server backups. Implement and
oversee automated backup procedures. Perform data recovery procedures. Work
in conjunction with DBA for database administration tasks. Performing
scheduled maintenance on all servers

* Inventory of hardware equipment. Handle software and hardware licenses,
warranties and purchase orders related to production, pre-production & test
servers at collocated server farm. Vendor liaison

* On call 24x7x365 for server and network outage issues. Troubleshoot and
identify source of downtime and work with engineering and Internet Service
Provider to correct the problems

* Constant researching of new tools and methods to better manage servers
and networking equipment. Deliver comprehensive alternatives and solutions
for server integration issues and network issues

* Troubleshoot server performance issues, identify bottlenecks and resolve
as necessary

* Manage corporate wide email system (MS Exchange). Perform maintenance and
backups of data

* Manage anti-virus programs and servers, keeps current with anti-virus
protection schemes & software
* Maintain current documentation of hardware and software and architecture
diagrams

Desired skills:
* Strong problem solving ability, good verbal and written communications
skills

* Solid understanding of networking computing environment concepts

* Understands principles of routing and server and network security

* Apache web server/ mod_perl

* Ability to script in shell/PERL

* Familiarity with MySQL, HTML
