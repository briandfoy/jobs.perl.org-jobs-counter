Description:
The DOE Joint Genome Institute in Walnut Creek, CA has an exciting Software
Developer opening available to support the Sequencing Quality Assurance /
Quality Control Group. Will develop software systems and analyze data to
support quality control of the JGI production sequencing platforms. Initial
work will focus on the development of a software system using LAMP for
analysis of short read sequence technology. Will also perform quality
control research and development work to investigate improved methods for
performing sequencing data analysis. Will work independently and
contributing as a member of a dynamic, multidisciplinary team environment.
Position reports to the QA/QC Group Lead.

integrate and deploy new and/or existing software applications and work
with other developers to coordinate work.
- Develop a software system for the analysis of short read sequence
technology.
- Create, manage, and query MySQL databases.
- Develop and utilize Perl classes, modules, and scripts for managing data
and performing analyses; and Perl/CGI web pages for providing user access.
- Interface with end-users to gather feedback and requirements.
- Provide responsive customer support; provide users with the prompt
resolution of data entry errors, software malfunctions, and other system
needs.
- Solve diverse problems by applying analytical skills and creativity.
- Communicate regularly with customers and supervisory staff about plans,
designs, and progress.
- Follow prescribed practices for software version control, testing,
documentation, and release engineering.
- Perform all assignments in accordance with ES&H, Security, and business
practice requirements and policies.
