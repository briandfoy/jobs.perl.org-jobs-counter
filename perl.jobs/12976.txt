The Manager, Software Development is responsible for assisting the CTO in
overseeing the developer group within the Information Technology

mentoring and guiding our PERL developers working out of the Oakville and
Ahmedabad (India) offices, as well as providing global standards on
development processes, code standards and build, test release mechanisms to
enable FlightNetwork.com’s delivery capability. As a technical leader, you
are responsible for driving quality and enabling technical innovation.

Responsibilities:

Specifically, the role’s responsibilities will include but are not limited
to:

•Establish and maintain global coding standards for PERL development
•Managing, developing and mentoring our global team of PERLdevelopers
•Create, implement and manage processes and ensure best practices for
software development, version control, unit testing etc. as well as build,
test, release mechanisms
•Work with PMO and the individual technical leads to track clear and
concise roadmaps, resulting in on-time delivery

Required skills:
•Bachelors in Computer Science or equivalent demonstrated experience with
strong underpinnings in algorithms and efficiency
•Substantial development experience in object oriented Perl code with
Moose,ORM systems such as DBIx::Class and the Catalyst framework (this is a
must) on Linux
•Experience with Perl web acceleration technologies such as mod_perl and
fastcgi on Apache and nginx
•Substantial web development experience, preferably with experience in a

•Knowledge of and experience in robust, scalable and secure coding

•10 years’ experience, with a minimum of 5 - 7 years as a PERL developer

deliver large scale software systems.
•Must be technically credible to lead teams of world-class software
engineers and be able to make the right design decisions in a Perl based
environment.
•Published code on CPAN (asset)
•Substantial experience developing and deploying production code in a Linux
based environment
