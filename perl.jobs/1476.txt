Reference Data Operations Support Analyst
 

A Reference Data Operations Support Analyst position involves a combination
of operations support and development responsibilities in a financial
services environment.  Responsibilities include loading feeds from vendors
into database applications on a predefined schedule.  Vendor feeds provide
data that must be validated, cross referenced against data already in the
applications and then loaded 

 

Responsibilities
Operations support responsibilities include:

Responding to problems communicated via email, telephone call, beep, error
logs, and reports generated by scheduled jobs within the team’s
responsibility.

Communicating with users and data vendors, problem logging, tracking and
escalation, incident reporting, and problem resolution (diagnose and
repair.).

 

Development responsibilities include:

Producing new reports, producing a new data loader for a new vendor feed,
modifying existing loaders to conform to new vendor specifications,
producing tools to monitor and improve referential data quality. 

Developing test plans, test scripts, test cases and expected results to
ensure that changes to the software infrastructure meet user requirements
and functional specifications.	

 

This position will also include on-call, shift work and weekend work.
